# Teacher Model Version History

本文件用于记录航空航天遥感目标检测项目中 Teacher 模型的版本迭代过程。

本项目当前阶段的核心目标不是直接训练最终部署模型，而是先构建高质量 Teacher 模型，用于后续伪标签生成、半监督学习和自适应学习。

## Version Overview

| Version | Name | Main Method | Purpose |
|---|---|---|---|
| v1.1.0 | 原图监督基线 Teacher | YOLOv8 原图训练 | 跑通监督学习训练流程，验证数据集与训练流程可行性 |
| v1.2.0 | SAHI 切图增强 Teacher | SAHI 滑窗切片 + YOLOv8 | 解决遥感小目标在原图缩放后特征丢失的问题，显著提升检测效果 |
| v1.3.0 | 最终伪标签生成 Teacher | 针对 vehicle 等短板类别优化 | 提升伪标签质量，为后续半监督学习和自适应学习做准备 |

## v1.1.0 - Original Image Supervised Baseline Teacher

v1.1.0 是本项目的初代 Teacher 模型。

该版本使用原始遥感图像进行 YOLOv8 监督训练，主要目标是跑通完整训练流程，包括数据集整理、标签转换、模型训练、指标评估和结果可视化。

该版本的意义在于验证了项目基础流程的可行性，但由于遥感图像中目标较小，原图缩放后小目标特征被压缩，导致检测效果偏弱。

### Key Findings

- 训练流程可以正常跑通
- 数据格式和标签转换基本可用
- 原图训练对小目标检测效果有限
- 小目标特征在缩放过程中容易丢失

## v1.2.0 - SAHI Slicing Enhanced Teacher

v1.2.0 是在 v1.1.0 基础上的改进 Teacher 模型。

该版本引入 SAHI 滑窗切片方法，将大尺寸遥感图像切分为局部图像块进行训练和检测，从而提高小目标在输入图像中的相对尺度，减少原图缩放带来的特征损失。

该版本相比 v1.1.0 检测指标实现明显提升，说明 SAHI 切图方法对遥感小目标检测是有效的。

### Key Improvements

- 使用 SAHI 切图增强小目标特征
- 显著提升遥感小目标检测效果
- 获得可用的初代高质量 Teacher 权重
- 为后续伪标签生成提供基础模型

### Remaining Problems

- vehicle 类别仍存在漏检问题
- 部分密集小目标定位不够准确
- 伪标签生成前仍需要进一步提升 Teacher 稳定性

## v1.3.0 - Final Pseudo-label Teacher

v1.3.0 是计划中的最终 Teacher 模型版本。

该版本将在 v1.2.0 的基础上，重点针对 vehicle 类别漏检、定位不准等问题进行优化。目标是得到一个更稳定、更适合批量生成伪标签的 Teacher 模型。

v1.3.0 的主要任务不是单纯追求训练指标，而是提高后续伪标签的可靠性，降低错误伪标签对 Student 模型和自适应学习过程的污染。

### Planned Focus

- 优化 vehicle 类别检测效果
- 提升密集小目标定位精度
- 筛选高质量伪标签
- 为半监督学习和自适应学习提供可靠 Teacher
